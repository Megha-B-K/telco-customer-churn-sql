# Telco Customer Churn Analysis – SQL

SQL-based analysis of **7,000+ telecom customers** to identify churn patterns, customer behavior, and revenue insights.

---

## Dataset

The dataset contains telecom customer information including:

- Customer demographics
- Contract types
- Monthly charges
- Internet service
- Payment methods
- Churn status

Dataset size: **7043 customers × 21 columns**

---

## SQL Analysis Performed

| Analysis | SQL Concept Used |
|--------|----------------|
| Total customers | COUNT() |
| Churn rate calculation | GROUP BY |
| Data cleaning | UPDATE |
| Churn by gender | GROUP BY |
| Senior citizen churn | Filtering + GROUP BY |
| Contract analysis | AVG() |
| Internet service churn | GROUP BY |
| Customer segmentation | CASE WHEN |
| Revenue analysis | SUM() |

---

## Key Business Insights

- Overall churn rate: **~27%**
- **Senior citizens churn more frequently**
- **Fiber optic customers show higher churn**
- **Month-to-month contracts have the highest churn**
- **Customers with tenure < 12 months are most likely to churn**
